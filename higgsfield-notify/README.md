# Higgsfield Generation Done Notifier (힉스필드 생성 완료 알림)

힉스필드([higgsfield.ai](https://higgsfield.ai)) Cinema Studio에서 **영상 생성이 끝나면** 알려주는 유저스크립트입니다. 완료 시점이 매번 달라 기다리기 번거로운 문제를 해결합니다.

- **화면 배너** (권한 불필요, 무조건 보임)
- **소리** (페이지를 한 번 클릭해 둔 뒤부터 재생됨 — 브라우저 자동재생 정책)
- **데스크톱 알림** (OS/브라우저 알림 허용 시)
- **탭 제목 ✅ 표시** (다른 탭에 있어도 눈에 띔)

> ⚠️ 비공식 도구입니다. Higgsfield 제작사와 무관합니다.

## 설치

1. [Tampermonkey](https://www.tampermonkey.net/) 설치
2. **"사용자 스크립트 허용(Allow User Scripts)" 켜기** (루트 [README](../README.md) 참고 — 안 켜면 실행 안 됨)
3. [`higgsfield-notify.user.js`](higgsfield-notify.user.js) 설치 → 힉스필드 탭 새로고침(F5)
4. 우측 상단에 `🎬 알림 실행됨 (진행중 N)` 배너가 뜨면 정상

## 동작 원리

생성 중인 타일에는 `data-job-status="queued"`/`"processing"` 같은 **작업 상태 속성**이 붙습니다.
이 "진행 중" 작업의 개수를 2초마다 세어, **개수가 줄어드는 순간(= 완료)** 에만 알림을 띄웁니다.
시작 시엔 개수가 늘어나므로 오탐이 없습니다.

## 문제 해결

- **배너조차 안 뜸** → "사용자 스크립트 허용"이 꺼져 있거나 스크립트가 비활성. F12 콘솔에 `[힉스알림] 시작, 진행중 = N` 로그 확인.
- **OS 알림만 안 뜸** → 윈도우 집중 지원 OFF, 브라우저 사이트 알림 "허용". (배너·소리는 정상이면 스크립트는 OK)
- **완료 감지가 안 됨** → 힉스필드가 `data-job-status` 속성을 바꾼 것. 생성 중 타일을 검사해 실제 속성을 확인한 뒤 스크립트의 `ACTIVE` 정규식/셀렉터를 수정.
