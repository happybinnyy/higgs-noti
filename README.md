# AI필드 (aifield)

AI 웹서비스를 편하게 쓰기 위한 **브라우저 유저스크립트 모음**입니다.
(AIfield — userscripts that make AI web apps nicer to use.)

> ⚠️ 모두 **비공식(unofficial)** 도구이며, 각 서비스 제작사와 아무 관련이 없습니다.
> 페이지 화면(DOM)만 읽어 편의 기능을 더할 뿐, 콘텐츠를 수집·재배포하거나 결제를 우회하지 않습니다.

## 설치 (공통)

1. 브라우저에 [Tampermonkey](https://www.tampermonkey.net/) 설치
2. ★ **"사용자 스크립트 허용(Allow User Scripts)" 켜기** — 최신 크롬/엣지/웨일 필수
   `확장 관리 → 개발자 모드 ON → Tampermonkey 세부정보 → 사용자 스크립트 허용 ON`
   (이게 꺼져 있으면 스크립트가 아예 실행되지 않습니다)
3. 원하는 도구 폴더의 `*.user.js` 를 설치 (Tampermonkey에서 열면 설치 화면이 뜸)
